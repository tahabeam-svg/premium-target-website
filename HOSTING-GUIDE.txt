=============================================
  Premium Target - Static Site Hosting Guide
  دليل رفع الموقع على هوستينجر
=============================================

الملفات:
--------
- index.html          -> الصفحة الرئيسية
- .htaccess           -> إعادة توجيه الروابط
- assets/             -> ملفات CSS و JavaScript والشعار
- images/             -> صور الموقع
- sitemap.xml         -> خريطة الموقع لجوجل
- favicon.png         -> أيقونة الموقع
- api/                -> ملفات البيانات + PHP
  - contact.php       -> نموذج التواصل (يرسل إيميل)
  - checkout.php      -> طلب الباقات (يحول لواتساب)
  - portfolio.json    -> بيانات الأعمال
  - packages.json     -> بيانات الباقات
  - blog.json         -> بيانات المدونة
  - testimonials.json -> بيانات الشهادات
  - portfolio/        -> تفاصيل كل عمل
  - blog/             -> تفاصيل كل مقال


خطوات الرفع على هوستينجر:
--------------------------

1. ادخل لوحة تحكم هوستينجر (hPanel)

2. اذهب لـ File Manager (مدير الملفات)

3. ادخل مجلد public_html

4. احذف كل الملفات الموجودة داخل public_html

5. ارفع كل محتويات مجلد static-site داخل public_html:
   - ارفع الملفات والمجلدات كما هي
   - تأكد من رفع ملف .htaccess (ملف مخفي)

6. تأكد أن الهيكل النهائي يكون:
   public_html/
   ├── .htaccess
   ├── index.html
   ├── favicon.png
   ├── sitemap.xml
   ├── assets/
   ├── images/
   └── api/
       ├── contact.php
       ├── checkout.php
       ├── portfolio.json
       ├── packages.json
       ├── blog.json
       ├── testimonials.json
       ├── portfolio/
       └── blog/

7. تحقق أن mod_rewrite مفعل (عادة مفعل تلقائياً في هوستينجر)

8. اضبط إعدادات PHP:
   - PHP Version: 8.0 أو أعلى
   - تأكد أن mail() مفعلة


إعداد الدومين:
--------------
- إذا عندك دومين premiumtarget.sa اربطه من DNS في هوستينجر
- غيّر الروابط في sitemap.xml إذا كان الدومين مختلف


ملاحظات مهمة:
-------------
- نموذج التواصل يستخدم PHP mail() لإرسال الإيميلات
- زر "ادفع الآن" في الباقات يحول العميل لواتساب مع رسالة جاهزة
- إذا أردت إضافة Stripe، ستحتاج استضافة تدعم Node.js
- لتعديل المحتوى (أعمال، مقالات، باقات): عدّل ملفات JSON في مجلد api/
- الموقع يعمل بثلاث لغات بدون مشاكل (عربي، إنجليزي، فرنسي)
